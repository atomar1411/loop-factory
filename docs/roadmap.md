# Roadmap

## V0.1 - Foundation

- Dual Codex and Claude Code plugin manifests.
- Core loop docs.
- Target-repo templates.
- Core skills.
- Claude Code agent profiles.
- One public `/loop-factory` surface for repo enablement and doctor checks.
- CLI `install`, `setup`, `doctor`, and optional automation commands.

## V0.2 - GitHub Loop Automation

- Richer `loop-factory issue create`.
- `loop-factory pr validate`.
- Label bootstrap.
- Task packet validation.
- PR body validation.
- Optional GitHub Actions examples for Claude Code and Codex-compatible workflows.

## V0.3 - Durable Orchestration

- Loop state reader from GitHub issues and PRs.
- Worktree inventory.
- Review and verifier report parsers.
- Local read-only dashboard.

## V0.4 - Policy Packs

- Auto-merge policy pack.
- Deploy-loop policy pack.
- Security-sensitive repo policy pack.
- Monorepo policy pack.

## V1.0 - Stable Open Source Release

- Versioned plugin release.
- Marketplace docs.
- Cross-surface examples.
- Backward-compatible template upgrade path.
